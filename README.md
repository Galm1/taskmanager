# Task Manager API

A REST API todo list application, built to sharpen my Java and Spring Boot skills.

## Tech Stack
- **Java**
- **Spring Boot**
- **PostgreSQL**
- **Maven**

## Purpose
A functional task manager where users can create, read, update and delete tasks.
